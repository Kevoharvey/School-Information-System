# Project Summary

The project is now organized as a professional Flask school system with:

- One shared CSS file: `static/css/style.css`
- One JavaScript file per page in `static/js/`
- Clean Bootstrap-based templates
- Database-backed routes for students, teachers, assignments, analytics, notifications, schedule, online registration, and AI assistant
- No placeholder operational data in `db.sql`
- First real registered account becomes admin

## Key Pages

- Landing page
- Online registration page
- Login and account pages
- Dashboard
- Students and student profile
- Teachers and subjects
- Assignments and submissions
- Analytics with Chart.js
- AI assistant
- Notifications
- Schedule
- Admin dashboard
